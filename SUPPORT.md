# Support

Need help with Nirman? Use the channels below.

## Community Support

- GitHub Discussions: use Q&A or General templates
- GitHub Issues: for confirmed bugs and feature requests

## Before Opening an Issue

Please check:

- Existing issues
- Existing discussions
- CONTRIBUTING guide
- README setup instructions

## What to Share for Faster Help

- Exact error message and logs
- Steps to reproduce
- Service involved (Gateway, Hub, Workers, Realtime, Dashboard, SDK)
- Environment details (OS, Docker, Node, Bun, Go, Python versions)

## Security Issues

For vulnerabilities, use private disclosure:

- https://github.com/deveshjhaq/nirman-baas/security/advisories/new
